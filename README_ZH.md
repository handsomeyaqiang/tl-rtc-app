### 安装步骤

1. 克隆仓库：

    ```bash
    git clone https://github.com/tl-open-source/tl-rtc-app.git
    ```
   
2. 进入项目目录：

    ```bash
    cd tl-rtc-app
    ```
   
3. 安装依赖：

    ```bash
    npm install
    ```

### 使用说明
1. 启动应用：

    ```bash
    npm start
    ```
   
2. 打开浏览器并访问 `http://localhost:9096`。
